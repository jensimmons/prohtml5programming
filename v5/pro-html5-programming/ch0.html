About This Interactive Book

Welcome to this interactive version of Pro HTML5 Programming. We teamed up with the folks at Apress to create this a new kind of book where we practice what we preach. We used a lot of the HTML5 APIs, like application cache, video, audio, canvas. We started with the same content as the printed and ebook versions, but then we took it to the next level. The book will reformat itself for different sized screens. You can edit the code in a live editor. And the book will work offline.
